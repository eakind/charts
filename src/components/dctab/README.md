### Tabs Attributes
| 参数 | 说明 |	类型 | 可选值 | 默认值 |
|---- | ---- | ---- | ---- | ---- |
| v-model | 绑定值，选中选项卡的 name | string | — | 第一个选项卡的 name |
| stretch | 标签的宽度是否自撑开 | boolean | — | true |
### Tab Item Attributes
| 参数 | 说明 |	类型 | 可选值 | 默认值 |
|---- | ---- | ---- | ---- | ---- |
| name | 唯一标志符 | string/number | — | — |
| title | 导航标题 | string | — | — |
